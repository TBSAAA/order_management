During my undergraduate studies in Computer Science, I majored in Systems Programming and Cyber Security. Throughout my time in university, I took a variety of courses that exposed me to a broad range of topics in computer science. Some of the key courses I took include:

1. Data Structures and Algorithms: This course focused on implementing data structures such as stacks, various types of linked lists, and binary trees using C++. For the course project, I developed a game called HexGame, which utilized algorithms such as backtracking and Monte Carlo simulations.

2. Systems Programming: This course delved into the implementation details of operating systems at a low level. For the course project, I used C to implement Linux commands like "cd" and "cp" as well as sockets.

3. Computer Organization: In this course, we studied the underlying logic of the MIPS chip, including its arithmetic logic unit and control unit. For the course project, I used assembly language in a simulator to implement higher-order functions and run complex programs on the MIPS architecture.

4. Network Security: This course covered various encryption algorithms, from the basic Caesar cipher to asymmetric encryption and digital signatures. For the course project, I implemented symmetric and asymmetric encryption algorithms using C and designed a process for using encryption algorithms and key exchange to secure network transmissions.

5. Computer Security: This course focused on protecting computers from attacks. For the course project, I conducted a scan of a target machine, identified vulnerabilities, and successfully attacked it to gain root access.

6. Web Development: This series of courses covered MySQL databases, PHP web development, and C# web development. For these courses, I developed a music website using PHP and a hotel booking and order management system using C# (.NET).

7. Mobile Development: This course covered the development of Android and iOS mobile applications. For the course project, I developed a to-do list app for both Android and iOS.

8. Internship Project: As part of my undergraduate studies, I collaborated with a group of classmates on an internship project for a startup company based in Brisbane, Australia. The company wanted to develop a flash card game website, and I took on the role of lead developer for the project. Over a period of three months, I taught myself Django and Bootstrap and successfully completed the development of the website.
